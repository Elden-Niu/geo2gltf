# 地理数据转GLTF工具 - 项目说明

## 📋 项目简介

这是一个用于将地理数据（GeoJSON 和 Shapefile）转换为 GLTF/GLB 格式的工具，支持单文件转换和批量转换，提供友好的图形化界面。

## 📁 项目结构

### 核心文件
```
shp2fbx/
├── geo2gltf.py                  # 核心转换器（支持 GeoJSON 和 SHP）
├── geo2gltf_gui_simple.py       # 图形界面（简洁版，推荐使用）
├── build_exe.py                 # 打包脚本（生成 .exe 文件）
├── setup.py                     # 安装配置文件
├── requirements.txt             # 依赖包列表
└── README.md                    # 项目说明文档
```

### 发布文件
```
dist/
└── 地理数据转GLTF工具.exe       # 打包后的可执行文件

地理数据转GLTF工具_v1.0/         # 发布版本目录
├── 地理数据转GLTF工具.exe
├── 使用说明.txt
├── logo.png
└── 示例文件/
```

## 🚀 功能特点

### 1. 单文件转换
- 支持 GeoJSON（.geojson, .json）和 Shapefile（.shp）格式
- 可自定义高度、颜色、透明度
- 支持自动缩放

### 2. 批量转换
- **自动扫描**：选择目录后自动扫描所有支持的文件
- **递归搜索**：自动包含子目录中的文件
- **独立配置**：可为每个文件单独设置颜色（双击文件列表项）
- **统一参数**：高度、透明度等参数统一应用

### 3. 图形界面特点
- 清爽简洁的界面设计
- 实时日志输出
- 进度条显示
- 批量转换统计信息

## 💻 使用方法

### 方式一：直接运行（需要 Python 环境）

1. 安装依赖：
```bash
pip install -r requirements.txt
```

2. 运行GUI：
```bash
python geo2gltf_gui_simple.py
```

### 方式二：使用可执行文件

直接运行 `地理数据转GLTF工具.exe`，无需安装 Python。

## 🔧 打包可执行文件

```bash
python build_exe.py
```

打包后的文件位于 `dist/地理数据转GLTF工具.exe`

## 📦 依赖库

- **pygltflib** - GLTF 文件读写
- **geopandas** - 地理数据处理
- **shapely** - 几何计算
- **fiona** - Shapefile 读取
- **mapbox-earcut** - 多边形三角化
- **numpy**, **pandas** - 数据处理

## 📝 批量转换使用流程

1. **选择输入目录** - 点击"浏览"按钮选择包含地理数据文件的目录
2. **自动扫描** - 系统自动扫描目录及子目录中的所有 GeoJSON 和 SHP 文件
3. **设置参数** - 调整高度值、颜色、透明度等参数
4. **配置单独颜色**（可选）- 双击文件列表中的文件可为其设置独立颜色
5. **开始转换** - 点击"开始批量转换"按钮
6. **查看日志** - 实时查看转换进度和结果

## 🎨 参数说明

- **高度值 (%)**: 控制3D模型的挤出高度，范围 1-50%
- **启用自动缩放**: 自动将坐标缩放到合适的范围
- **默认颜色**: 设置模型的颜色（可为每个文件单独配置）
- **透明度**: 设置模型的透明度，范围 0-1
