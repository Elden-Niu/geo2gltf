# 现代化GUI使用指南

## 🎨 全新iOS风格界面

`geo2gltf_gui_modern.py` 是全新设计的现代化界面版本，具有以下特点：

### ✨ 主要特性

1. **🍎 iOS风格设计**
   - 清新的浅色背景和卡片式布局
   - 圆角按钮和现代化字体
   - 悬停效果和流畅动画
   - 统一的配色方案（iOS蓝、绿、紫等）

2. **🖼️ 自定义LOGO**
   - 在界面顶部显示您的logo
   - logo文件：`logo.png`（自动加载）
   - 窗口图标也使用logo

3. **🎨 每个文件独立配置颜色**
   - 批量转换时可为每个文件设置不同颜色
   - 双击文件列表中的文件名即可配置颜色
   - 文件列表会用不同颜色显示，方便识别

4. **📦 卡片式布局**
   - 功能分区清晰
   - 白色卡片容器
   - 优雅的间距和对齐

---

## 💼 单文件转换模式

### 操作步骤

1. **选择文件**
   - 点击"文件选择"卡片中的"浏览"按钮
   - 选择GeoJSON或Shapefile文件
   - 输出目录会自动设置

2. **调整参数**
   - **高度值**：拖动滑块调整（1-50%）
   - **启用自动缩放**：推荐勾选
   - **默认颜色**：点击"选择"按钮选择颜色
   - **透明度**：拖动滑块调整（0-1）

3. **开始转换**
   - 点击大的蓝色"开始转换"按钮
   - 在转换日志中查看进度
   - 完成后会弹出提示框

### 界面特点
- 圆角的蓝色按钮，鼠标悬停时会变亮
- 清晰的卡片分区
- 实时日志显示在浅灰色文本框中

## 🎨 批量转换模式（独立颜色配置）

### 操作步骤

#### 第1步：选择目录
- **输入目录**：包含待转换文件的目录
- **输出目录**：转换结果保存位置
- 使用"浏览"按钮选择目录

#### 第2步：配置选项
- ☑ **递归搜索子目录**：处理所有子目录
- **文件类型**：选择GeoJSON、Shapefile或两者
- 点击紫色"扫描文件"按钮

#### 第3步：配置文件颜色（核心功能！）

扫描完成后，文件列表会显示所有找到的文件：

**方法1：双击配置**
1. 双击文件列表中的任意文件名
2. 在弹出的颜色选择器中选择该文件的颜色
3. 点击确定

**方法2：批量使用默认颜色**
- 所有文件默认使用"默认参数设置"中的颜色
- 只为需要特殊颜色的文件单独配置

**颜色标记**
- 配置颜色后，文件名会以该颜色显示
- 方便识别哪些文件有自定义颜色

#### 第4步：调整默认参数
- 与单文件模式相同的参数设置
- 未单独配置颜色的文件将使用默认颜色

#### 第5步：开始批量转换
- 点击绿色"开始批量转换"按钮
- 观察实时进度：
  - 进度标签显示当前文件
  - 进度条显示总体进度
  - 日志窗口显示详细信息
- 完成后显示统计报告

## 🎨 独立颜色配置示例

### 场景：广汉市行政区域批量转换

假设有以下文件：
```
广汉行政区域/
├── 三星堆镇.geojson   → 想用 红色
├── 三水镇.geojson     → 想用 绿色
├── 南丰镇.geojson     → 想用 蓝色
├── 向阳镇.geojson     → 使用默认颜色
└── 小汉镇.geojson     → 使用默认颜色
```

**操作步骤：**

1. 选择输入目录：`C:\Data\广汉行政区域\`
2. 点击"扫描文件"，找到5个文件
3. 双击"三星堆镇.geojson"，选择红色
4. 双击"三水镇.geojson"，选择绿色  
5. 双击"南丰镇.geojson"，选择蓝色
6. 向阳镇和小汉镇不配置，使用默认颜色
7. 点击"开始批量转换"

**结果：**
- 三星堆镇.gltf → 红色模型 🔴
- 三水镇.gltf → 绿色模型 🟢
- 南丰镇.gltf → 蓝色模型 🔵
- 向阳镇.gltf → 默认颜色模型
- 小汉镇.gltf → 默认颜色模型

## 💡 使用技巧

### 技巧1：快速批量转换
如果所有文件使用相同颜色：
1. 在"默认参数设置"中设置颜色
2. 扫描文件后，直接点击"开始批量转换"
3. 无需单独配置每个文件

### 技巧2：重置颜色
如果想重置文件颜色：
1. 修改"默认参数设置"中的默认颜色
2. 重新点击"扫描文件"
3. 所有文件颜色将重置为新的默认颜色

### 技巧3：按区域配色
对于地理数据：
- 市区用蓝色
- 工业区用灰色
- 公园用绿色
- 商业区用橙色

### 技巧4：参数实时同步
- 高度、透明度等参数在两个标签页之间自动同步
- 在任一标签页调整参数，另一个标签页也会更新

## 🐛 常见问题

### Q1: LOGO不显示？
**A:** 
1. 确保`logo.png`文件存在于工具目录
2. 确保已安装Pillow：`pip install Pillow`
3. 检查logo文件格式是否正确（PNG格式）

### Q2: 如何批量使用相同颜色？
**A:** 在"默认参数设置"中设置颜色，扫描文件后直接转换即可。

### Q3: 配置颜色后想取消？
**A:** 重新扫描文件，或关闭并重新打开GUI。

### Q4: 文件列表颜色显示不明显？
**A:** 这是正常的，列表中的颜色用于标记，实际输出的3D模型颜色会更明显。

### Q5: 可以为同一文件配置多次颜色吗？
**A:** 可以，双击文件名重新选择颜色即可覆盖之前的设置。

## 🎯 最佳实践

### 1. Logo准备
- 推荐尺寸：80x80像素或更大
- 格式：PNG（支持透明背景）
- 命名：logo.png
- 位置：与GUI脚本同目录

### 2. 批量转换流程
```
选择目录 → 扫描文件 → 配置特殊颜色 → 设置默认参数 → 开始转换
```

### 3. 颜色配置策略
- 重要区域：显眼颜色（红、橙）
- 普通区域：中性颜色（蓝、绿）
- 背景区域：浅色（浅灰、浅蓝）

### 4. 性能优化
- 大量文件（>100个）时，建议分批处理
- 递归模式会搜索所有子目录，文件多时会较慢
